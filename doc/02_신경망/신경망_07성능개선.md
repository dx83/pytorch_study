# 성능 개선 기법

### 과적합 : 학습 데이터에 너무 잘 맞춰줘서 테스트 데이터에서 성능이 떨어질 수 있음

### 과적합 방지 기법
- 조기 종류 (Early Stopping) : 검증 데이터의 손실(loss)이 일정 횟수 이상 감소하지 않으면 학습을 조기 종료하여 과적합을 방지
- 정규화 (Regularization) : L1 및 L2 정규화를 활용하여 모델의 가중치를 제한함으로써 과적합을 줄일 수 있다.
- ReduceLROnPlateau : 성능 정체시 학습률 감소 스케줄러
```python
from torch.optim.lr_scheduler import ReduceLROnPlateau

scheduler = ReduceLROnPlateau(optimizer, mode='min', patience=3, factor=0.1)
for epoch in range(50):
    train_loss = 0.2 * (50 - epoch)
    scheduler.step(train_loss)
# loss 가 더 이상 줄지 않을 때 learning rate를 낮춰서 학습을 더 안정적으로 이어가게 해줌
```

### 배치 정규화 (Batch Normalization) 적용
- 각 층의 출력 분포를 미니배치 기준으로 정리해서, 모델이 더 빠르고 안정적으로 학습되도록 도와주는 기법
    - 학습이 더 안정적
    - 더 큰 학습률을 써도 잘 버팀
    - 초기값에 덜 민감
    - 약간의 regularization 효과가 있음
    - 깊은 신경망 학습에 도움

- BatchNorm
    - 학습 모드 : 현재 미니배치의 평균과 분산을 사용
    - 평가 모드 : 학습 중 누적한 running mean, running variance를 사용
        - running mean = 학습 중 관찰한 평균의 이동 평균
        - running variance = 학습 중 관찰한 분산의 이동 평균
        - 검증/테스트 전에 model.eval()을 해야함

```python
class MLP(nn.Module):
    def __init__(self):
        super(MLP, self).__init__()
        self.fc1 = nn.Linear(28*28, 128)
        self.bn1 = nn.BatchNorm1d(128)
        self.fc2 = nn.Linear(128, 64)
        self.bn2 = nn.BatchNorm1d(64)
        self.fc3 = nn.Linear(64, 10)

    def forward(self, x):
        x = x.view(-1, 28*28)
        x = F.relu(self.bn1(self.fc1(x)))
        x = F.relu(self.bn2(self.fc2(x)))
        x = self.fc3(x)
        return x
```

### 드롭아웃 (Dropout) 활용
- 과적합을 막기 위해 일부 뉴런을 랜덤하게 비활성화하여 모델의 일반화 성능을 높인다.
    - 특정 뉴런에 대한 의존도 감소
    - 여러 작은 모델을 앙상블하는 것과 비슷한 효과
    - 일반화 성능 향상
    - 과적합 완화

```python
class MLP(nn.Module):
    def __init__(self):
        super(MLP, self).__init__()
        self.fc1 = nn.Linear(28*28, 128)
        self.dropout1 = nn.Dropout(0.5)     # 끄는 비율
        self.fc2 = nn.Linear(128, 64)
        self.dropout2 = nn.Dropout(0.5)     # 보통 0.2 ~ 0.5
        self.fc3 = nn.Linear(64, 10)
    
    def forward(self, x):
        x = x.view(-1, 28*28)
        x = F.relu(self.dropout1(self.fc1(x)))
        x = F.relu(self.dropout2(self.fc2(x)))
        x = self.fc3(x)
        return x
```
