# 역전파 (Backward Pass, Backpropagation)
`손실 함수로부터 계산된 오차를 기반으로, 모델의 각 파라미터(가중치, 편향)가 출력에 미치는 영향을 계산하는 과정`

- 이 과정은 수학의 연쇄 법칙을 사용하여 모델의 모든 연산 단계에서 각 파라미터의 기울기를 구한다.
- 이 기울기는 이후 파라미터 업데이트 단계에서 옵티마이저에 의해 사용되어 모델의 예측이 실제 정답에 더 가깝도록 가중치를 조정하는 데 중요한 역할

<br>

1. 오차 계산: 예측값 - 실제값
2. 책임 추적: 어떤 파라미터(가중치/편향)가 얼마나 영향을 미쳤는지 계산
3. 수정: 잘못된 정도만큼 파라미터를 조정하여 예측 개선

<br>

### 역전파와 연쇄 법칙 (Chain Rule)
- 연쇄 법칙은 복잡한 함수의 미분을 계산할 때 각 부분의 미분 값을 곱해 전체 미분 값을 구하는 수학적 원리다.

$$\frac{dy}{dx} = \frac{dy}{dg} \times \frac{dg}{dx}$$

> dg/dx : x가 변하면 중간값 g가 얼마나 변하는가  
> dy/dg : g가 변하면 최종값 y가 얼마나 변하는가  
> dy/dx : 결국 x가 변하면 y가 얼마나 변하는가  

- 앞 단계가 변할 때 중간값이 얼마나 변하는지 × 중간값이 변할 때 최종값이 얼마나 변하는지를 곱하는 것입니다.

### 역전파 과정
1. 기울기 계산의 원리
    - 모델의 출력과 실제 정답 간의 차이는 손실 함수로 계산
    - 역전파는 손실값이 모델의 각 파라미터에 어떤 영향을 미치는지 계산
    - 각 계층의 연산은 미분의 연쇄 법칙을 통해 연결되며, 최종으로 모든 계층의 기울기가 계산됨

2. 자동 미분 (Autograd)활용
    - requires_grad=True 로 설정된 텐서에 대해 모든 연산 기록을 자동으로 저장
    - loss.backward()를 호촐하면 저장된 연산 기록을 기반으로 각 파라미터의 기울기를 자동으로 계산
    - 복잡한 수학적 미분 계산을 직접 구현하지 않고도 모델 학습에 필요한 기울기 계산을  쉽게 수행

3. 중요 사항: 기울기 초기화
    - 학습 단계마다 optimizer.zero_grad()를 호출하여 이전 단계에서 계산된 기울기를 초기화해야 한다.
    - 기울기를 초기화하지 않으면 각 학습 단계의 기울기가 누적되어 잘못된 가중치 업데이트가 이루어질 수 있다.

<br>

### 예시
```python
import torch
import torch.nn as nn
import torch.optim as optim

# 선형 회귀 모델 정의: y = Wx + b
class LinearRegressionModel(nn.Module):
    def __init__(self, input_dim, output_dim):
        super(LinearRegressionModel, self).__init__()
        self.linear = nn.Linear(input_dim, output_dim)
    
    def forward(self, x):
        return self.linear(x)

# 예제 데이터
x = torch.tensor([[1.0], [2.0], [3.0], [4.0]])
y_true = 2* x + 1

# 모델, 손실 함수, 옵티마이저 설정
model = LinearRegressionModel(1, 1)
criterion = nn.MSELoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)

# 학습 과정: 100 에포크 동안 파라미터 업데이트 수행
for epoch in range(100):
    optimizer.zero_grad()           # 이전 기울기 초기화
    pred = model(x)                 # 순전파: 모델 예측값 계산
    loss = criterion(pred, y_true)  # 손실 계산: 예측과 실제 값 간 차이 측정
    loss.backward()         # 역전파: 손실을 각 파라미터로 전파하여 기울기 계산
    optimizer.step()        # 파라미터 업데이트: 기울기와 학습률 이용해 가중치 조정

    if (epoch + 1) % 20 == 0:
        print(f"Epoch {epoch+1}, Loss: {loss.item():.4f}")
```

<br>

### 옵티마이저의 업데이트 방식
- SGD (Stochastic Gradient Descent)
    - 가장 기본적인 옵티마이저
    - 각 파라미터를 기울기와 학습률에 따라 단순하게 업데이트

- Adam (Adaptive Moment Estimation)
     - 모멘텀과 적응적 학습률 개념을 도입해 기울기의 1차 모멘트(평균)와 2차 모멘트(분산)를 사용해 파라미터를 업데이트한다.
     - Adam은 특히 복잡한 문제에서 빠른 수렴과 안정성을 보여준다.

- RMSProp
    - 학습률을 파라미터마다 조정하며, 기울기 제곱의 이동 평균을 활용해 업데이트하는 방식
