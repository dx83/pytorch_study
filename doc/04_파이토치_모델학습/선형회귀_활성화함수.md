# 선형 회귀 모델과 활성화 함수

### 선형 회귀 모델 (Linear Regression Model)
- 가장 단순한 형태의 예측 모델
- 입력 x가 일정하게 변하면, 출력 y도 일정한 규칙으로 변하는 관계를 나타냅니다.
- 모델은 주어진 입력 x에 대해 $y = W \cdot x + b$ 의 형태로 예측값 y를 계산한다.
    - W (가중치, Weight) : 입력 데이터에 곱해지는 계수로 입력값이 출력에 미치는 영향을 조절
    - b (편향, Bias) : 모델의 예측 결과에 일정한 값을 더하는 역할을 하며, 데이터가 원점을 지나지 않을 때 보정
- 주택 가격 예측, 키와 몸무게 간의 관계, 공부와 성적 간 관계 등에 사용

<br>

### 활성화 함수 (Activation Function)
- 신경망에서는 비선형 활성화 함수를 도입해 각 층의 선형 변환 후에 비선형 변환을 적용한다. 이를 통해 신경망은 입력과 출력 사이의 복잡한 관계를 모델링할 수 있으며, 여러 층을 쌓아 심층 구조를 형성할 때 중요한 역할을 한다.

- 활성화 함수의 주요 역할
    1) 빈선형성 도입 : 각 레이어의 출력에 비선형성을 추가해 신경망이 단순 성형 모델보다 훨씬 더 복잡한 함수와 관계를 학습할 수 있도록 한다.
    2) 표현력 확장 : 비선형 활성화 함수 덕분에 여러 층을 쌓아 구성한 딥러닝 모델은 입력 데이터를 다양한 방식으로 변환하여 고차우너의 특징 공간으로 매핑할 수 있다.
    3) 모델 학습 안정성 및 효율성 : 적절한 활성화 함수의 선택은 모델의 학습 속도와 성능에 큰 영향을 미치며, 특정 활성화 함수는 기울기 소실 문제를 완화하는 데 도움을 줄 수 있다.

- 종류
    1) 시그모이드 (Sigmoid) 함수
    $$\sigma(x) = \frac{1}{1 + e^{-x}}$$
    - 출력 범위가 (0, 1)로 압축되며, 확률 해석을 할 수 있지만 큰 입력값에서 기울기 소실 문제가 발생할 수 있다.

    2) 하이퍼블릭 탄젠트 (Hyperblic Tangent, Tanh) 함수
    $$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$
    - 출력 범위가 (-1,1)로 압축되어 시그모이트보다 중심이 0에 가까워 기울기 소실 문제를 다소 완화할 수 있다.

    3) 렐루 (Rectified Linear Unit, ReLU) 함수
    $$\mathrm{ReLU}(x) = \max(0, x)$$
    - 계산이 간단하고 빠르며, 음수 값을 0으로 만들어 모델에 비선형성을 도입한다. 그러나 일부의 경우 '죽은 ReLU' 문제가 발생할 수 있다.

### 기울기 소실 (Vanishing Gradient)
- 기울기가 너무 작아지면(즉, 거의 0에 가까워지면) 가중치 업데이트가 제대로 이루어지지 않아 학습이 정체되는 현상

- 원인
    1) 딥러닝 모델의 계측이 깊어질수록 역전파 과정에서 기울기가 여러 계층을 거치며 연속해 곱해진다. 이 과정에서 기울기가 지수적으로 감소할 수 있어 초기 층에서는 거의 0에 가까운 기울기만 전달된다.
    2) 활성화 함수인 Sigmoid와 Tanh은 입력값이 매우 크거나 작을 때 기울기가 급격하게 작아지는 특성이 있다.
    
- 영향
    1) 기울기가 소실되면 가중치 업데이트가 거의 이루어지지 않아 모델의 학습 속도가 매우 느려지거나 학습이 아예 멈출 수 있다.
    2) 충분한 학습이 이루어지지 않아 모델이 데이터의 복잡한 패턴을 제대로 학습하지 못하게 되어 최종적으로 모델의 성능이 낮아진다.

- 해결
    1) 활성화 함수 변경
    2) 잔차 연결(Residual Connection)이나 배치 정규화(Batch Normalization) 기법 도입
