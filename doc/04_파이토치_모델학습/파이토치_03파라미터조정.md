# 파라미터 업데이트 (Parameter Update)
`딥러닝 모델 학습의 마지막 단계인 파라미터 업데이트는 역전파 과정에서 계산된 기울기(Gradient)를 이용해 모델의 각 파라미터(가중치와 편향)를 조정하는 단계이다.`

- 이 과정을 통해 모델은 입력 데이텅에 대한 예측 오류를 줄여나가며, 반복 학습을 통해 점차 더 나은 성능을 발휘한다.

<br>

- 옵티마이저의 역할
    - 역전파 과정에서 계산된 기울기를 기반으로, 각 파라미터를 업데이트하는 알고리즘

- 업데이트 방식
    - 가중치 업데이트는 일반적으로 파라미터의 새로운 값은 이전 값에서 학습률과 해당 파라미터의 기울기 곱만큼을 빼는 방식으로 업데이트된다.
    - 업데이트는 모델이 예측 오류를 줄이도록 유도하며, 배치 혹은 에포크마다 반복해서 수행된다.
    - 중요한 점은 학습 단계마다 이전 단계에서 누적된 기울기를 초기화해야 한다. 그렇지 않으면 기울기 누적되어 잘못된 방향으로 업데이트가 이루어질 수 있다. 이를 위해 optimizer.zero_grad()를 사용

- 반복 학습
    - 전체 데이터셋을 여러번 반복(순전파, 손실 계산, 역전파, 파라미터 업데이트)하면서 점진적으로 손실값을 줄이고 모델의 성능을 개선한다.
    - 반복 업데이트를 통해 모델은 입력 데이터의 복잡한 패턴을 점차 학습하게 된다.
