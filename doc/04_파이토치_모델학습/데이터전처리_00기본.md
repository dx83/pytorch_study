## 정규화 (Normalization)
- 이미지의 픽셀값이나 기타 수치형 데이터를 일정 범위로 조정하는 작업
- 정규화를 통해 각 입력 특성의 스케일을 맞추면 모델 학습 과정에서 각 입력값이 균등하게 영향을 주어 학습이 안정되고 빠르게 수렴할 수 있기 때문이다.

## 데이터 증강 (Data Augmentation)
- 원본 데이터를 회전, 이동, 크기 조절, 좌우 반전 등 다양한 변환을 적용해 새로운 학습 데이터를 생성하는 작업
- 데이터 증강은 학습 데이터의 다양성을 확보하여 모델이 다양한 상황에 대해 일반화할 수 있도록 도와준다.

## 데이터 분할 (Data Splitting)
- 전체 데이터셋을 학습, 검증, 테스트 데이터로 나누어 모델의 학습과 일반화 성능을 평가
- 학습 데이터로 모델을 훈련시키고, 검증 데이터로 하이퍼파라미터를 조정한 후 최종 테스트 데이터셋으로 모델의 성능을 객관적으로 평가

<br>

```python
from torchvision import datasets, transforms

transform = transforms.Compose([                # 여러 개의 이미지 변환 작업을 하나로 묶음
    transforms.RandomRotation(10),              # -10 ~ 10 도 사이에서 랜덤하게 회전
    transforms.ToTensor(),                      # 이미지를 PyTorch 텐서로 변환
    transforms.Normalize((0.1307,), (0.3081,))  # 정규화
])
train_dataset = datasets.MNIST(root='./data', train=True, download=True, transform=transform)
```
- .Normalize((0.1307,), (0.3081,)) : MNIST 데이터셋의 평균과 표준편차
    - 정규화된 값 = (픽셀값 - 평균) / 표준 편차
    - MNIST는 흑백 이미지라 채널이 1개이므로 (0.1307,), (0.3081,) 처럼 값이 하나씩 들어갑니다.
