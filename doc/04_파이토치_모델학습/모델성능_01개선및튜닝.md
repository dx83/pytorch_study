# 모델 개선 및 하이퍼파라미터 튜닝

### 모델 개선을 위한 주요 기법

1. 정규화 (Reqularization) 기법
- 드롭아웃 (Dropout) : 신경망 학습 시 일부 뉴련을 임의로 끄는 방식으로, 과적합을 방지하고 모델의 일반화 성능을 높인다.
- 가중치 감소 (Weight Decay) : 가중치 크기를 제한해 모델이 과도하게 복잡해지는 것을 방지

<br>

2. 데이터 증가 (Data Augmentation)
- 학습 데이터에 회전, 이동, 크기 조절, 좌우 반전 등 다양한 변환을 적용해 데이터의 다양성을 확보하고, 모델이 다양한 상황에 대해 학습할 수 있도록 한다.

<br>

3. 하이퍼파라미터 튜닝
- 학습률 (Learning Rate), 배치 크기 (Batch Size), 에포크 (Epoch) 수 등 여러 하이퍼파라미터를 실험하며, 최적의 모델 성능을 도출한다. 이때 최적의 하이퍼파라미터 조합을 찾기 격자 탐색 (Grid Search), 무작위 탐색 (Random Search) 혹은 베이지안 최적화 (Bayesian Optimization)와 같은 방법을 사용할 수 있다.

#### 격자 탐색 Grid Search
- 미리 정한 하이퍼파라미터 후보들을 모든 조합으로 실험하는 방법이다.
```python
learning_rate = [0.1, 0.01, 0.001]
batch_size = [32, 64, 128]
위 경우 총 9가지 조합을 모두 실험한다.
### 장점
- 구현이 쉽다.
- 모든 조합을 빠짐없이 확인한다.
### 단점
- 조합 수가 많아지면 시간이 매우 오래 걸린다.
- 중요하지 않은 파라미터까지 모두 탐색하므로 비효율적일 수 있다.
```

#### 무작위 탐색 Random Search
- 정해진 범위 안에서 하이퍼파라미터 조합을 무작위로 뽑아 실험하는 방법이다.
```python
learning_rate = 0.0001 ~ 0.1 사이에서 랜덤 선택
batch_size = [32, 64, 128] 중 랜덤 선택
### 장점
- 격자 탐색보다 빠르게 좋은 조합을 찾을 수 있다.
- 중요한 파라미터가 일부일 때 효율적이다.
### 단점
- 무작위라서 결과가 실행할 때마다 달라질 수 있다.
- 최적 조합을 반드시 찾는다는 보장은 없다.
```

#### 베이지안 최적화 Bayesian Optimization
- 이전 실험 결과를 바탕으로 다음에 실험할 하이퍼파라미터 조합을 똑똑하게 선택하는 방법이다.
- 즉, 성능이 좋을 가능성이 높은 영역을 우선적으로 탐색한다.
```python
### 장점
- 적은 실험 횟수로 좋은 조합을 찾을 가능성이 높다.
- 학습 시간이 오래 걸리는 모델에 유리하다.
### 단점
- 구현이 격자 탐색이나 무작위 탐색보다 복잡하다.
- 초기 설정이나 라이브러리 사용법을 알아야 한다.
```
