### 텐서 (Tensor)
- 다차원 배열 형태의 데이터 구조
- 숫자 데이터를 저장하고 연산하는 기본 단위
- 스칼라, 벡터, 행렬을 일반화하여 3차원 이상의 배열까지 포괄하는 개념

### xpu
- 설치법
```
pip uninstall torch torchvision torchaudio intel-extension-for-pytorch -y
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/xpu
```

- 가능 여부 확인 코드
```
import torch

print(torch.xpu.is_available())         # True
print(torch.xpu.get_device_name(0))     # Intel(R) Arc(TM) Graphics

device = torch.device("xpu")
x = torch.randn(3, 3).to(device)
print(x)
```

### 텐서의 핵심 속성
- Shape : 모양
- Data Type : 자료형
- Device : 텐서가 저장된 위치 (CPU/GPU/XPU)

<br>

- `requires_grad=True` : x에 대해 미분을 추적
- `y.backward()` : 미분 계산 실행

<br>

- torch.zeros((a, b)) : 모든 값이 0인 a × b 행렬 텐서 생성
- torch.ones((a, b)) : 모든 값이 1인 a × b 행렬 텐서 생성
- torch.full((a, b), c) : 모든 값이 c인  a × b 행렬 텐서 생성
- torch.arange(a, b, step=c) : a부터 b까지 (b제외) c씩 증가하는 값을 만든 텐서
- torch.linspace(a, b, steps=c) : a부터 b까지를 동일한 간격으로 c개로 나눈 텐서

### 연산
`브로드캐스팅` : 크기가 서로 다른 행렬(또는 텐서)끼리 산술 연산을 할 수 있도록, 부족한 차원을 컴퓨터가 자동으로 확장해 주는 기능

```
a = torch.tensor([1,2,3])   # shape (3,)
b = torch.tensor([10])      # shape (1,)
# b가 자동으로 [10, 10, 10]으로 확장
print(f"a + b = {a + b}")   # tensor([11, 12, 13])
```

### 인덱싱, 슬라이싱, 불리언 마스킹, reshape
- 텐서[행 인덱스, 열 인덱스] : 특정 위치의 원소
- 텐서[a:, :] : 콜론(:)을 사용하여 여러 행이나 열을 동시에 선택
- mask = 텐서 > 50 : 텐서에 조건을 부여하면 True/False 값이 담긴 마스크 텐서가 생성
- 차원 변경(reshape) : 데이터의 형태(차원)를 변경하여 모델에 입력하기 적합한 형태로 변환

<br>

### torch.
- torch.cat : 특정 차원을 기준으로 텐서 이어 붙이기
- torch.stack : 특정 차원을 만들어 텐서를 쌓음
- torch.split(대상, split_size_or_sections=x) : x 크기만큼씩 텐서를 잘라서 리스트로 반환
- torch.chunk(대상, chunks=x) : 텐서를 x개로 나눈다.
