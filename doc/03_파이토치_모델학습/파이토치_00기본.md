# 모델 학습

### 순전파 (Forward Pass)
- 입력 데이터를 모델의 각 계층(Layer)을 거쳐 최종 예측 결과를 도출하는 과정
- 입력 데이터가 모델의 연산 흐름을 따라 이동하며, 예측값을 생성하는 전과정

<br>

1. 입력 처리 (Input Processing)
- 데이터가 모델이 이해할 수 있는 형태로 변환하기 위해 전처리 작업을 수행하여 입력 텐서로 만든다.

2. 계층별 연산 (Layer-Wise Computation)
- 모델의 각 계층에서는 선형 변환이 먼저 수행
- 이어스 각 계측에서는 비선형 활성화 함수를 적용해 데이터의 복잡한 패턴을 학습할 수 있도록 한다.
- 이 과정을 통해 입력 데이터가 점차 고차우너 특징의 공간으로 변환된다.

3. 출력 생성 (Output Generation)
- 마지막 계측에서는 최종 예측 결과가 산출
