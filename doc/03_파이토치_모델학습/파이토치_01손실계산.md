# 손실 계산
`모델이 예측한 값과 실제 정답 사이의 차이를 측정하는 단계`

- 손실 함수를 사용해 모델의 오차를 수치로 나타내며, 이 값이 클수록 모델의 예측이 실제와 많이 다르다는 것을 의미
- 학습의 최종 목표는 이 손실값을 최소화하는 방향으로 모델의 파라미터(가중치와 편향)를 조정하는 것이다.

<br>

1. 크로스 엔트로피 손실 (Cross-Entropy Loss)
    - 분류 문제에서 자주 사용
    - 실제 클래스 (정답 라멜)에 높은 확률이 할당되도록 모델을 학습
    - 이 함수는 로그 손실을 기반으로 하며, 모델이 잘못된 예측을 할 경우 큰 페널티 부여

2. MSE (Mean Squared Error, 평균 제곱 오차)
    - 회귀 문제에서 자주 사용
    - 예측값과 실제값의 차이를 제곱한 뒤 평균을 구해 반환
    - 제곱을 취하기 때문에 큰 오차에 더 큰 벌점을 주며, 오차를 최소화하는 방향으로 모델을 학습

3. MAE (Mean Absolute Error, 평균 절대 오차)
    - 예측값과 실제값 차이의 절대값을 평균해 계산하며, 이상치에 덜 민감한 특성이 있다.

<br>

### 손실 함수 선택 기준
- 문제의 특성 (분류 vs 회귀)
- 모델의 출력 형태
- 데이터의 특성 및 분포

`손실 함수는 모델 학습의 방향을 결정하는 중요한 요소이므 문제에 적합한 손실 함수를 선택하는 것이 성능에 큰 영향을 미친다.`

<br>

### 손실 계산 과정
1. 손실 함수 선택
    - 분류 문제 : 크로스 엔트로피 손실
    - 회귀 문제 : MSE, MAE

2. 손실값의 의미
    - 모델이 얼마나 잘못 예측했는지를 나타내는 지표
    - 값을 줄이는 것이 학습의 목표
    - 손실 함수를 통해 모델의 오차를 정량화하면 옵티마이저는 이를 바탕으로 가중치를 조정하여 오차를 줄이는 방향으로 학습을 진행

3. 학습 목표
    - 순전파와 역전파를 통해 손실값이 점차 감소하도록 파라미터를 업데이트
        1) 순전파: 모델에 입력 데이터를 통과시켜 예측값 산출
        2) 손실 계산: 예측값과 실제값 간의 오차를 손실 함수를 통해 측정
        3) 역전파: 손실 함수의 값을 기준으로 각 파라미터의 기울기를 계산하고, 이를 통해 모델의 가중치를 업데이트
        4) 이 과정이 반복되면 모델은 점차 올바른 예측을 할 수 있도록 학습된다.
