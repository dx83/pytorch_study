# 합성곱 레이어 (Convolution Layer)

### 필터(커널)의 개념과 동작 원리
- 입력 데이터에 필터(또는 커널)를 적용하여 특징을 추출
- 이 필터는 입력 이미지 위를 슬라이딩하면서 각 위치에 요소별 곱셉과 덧셈 연산 (합성곱 연산)을 수행

<br>

```python
[1, 0, -1]
[1, 0, -1]
[1, 0, -1]
```
- 어두운 영역에서 밝은 영역으로 변할 때 (왼쪽에서 오른쪽으로) 음수 값이 생성
- 밝은 영역에서 어두운 영역으로 변할 때 (오른쪽에서 왼쪽으로) 양수 값이 생성

<br>

### 스트라이드 (Stride)와 패딩 (Padding)
- 스트라이드 (Stride) : 필터가 이동하는 간격, 스트라이드가 커질수록 출력 특성 맵의 크기는 작아진다.
- 패딩 (Padding) : 입력 이미지의 가장자리에 추가되는 픽섹을 의미
    - 'valid' : 패딩 없음, 출력 크기 작아짐
    - 'same' : 출력 크기가 입력과 동일하도록 패딩을 추가

$$출력 크기 = \frac{입력 크기 - 필터 크기 + 2 \times 패딩}{스트라이드} + 1$$

<br>

### 채널의 개념과 다중 채널 처리
- 합성곱 신경망에서는 각 채널을 독립적으로 처리한 후 그 결과를 합산하여 하나의 특성 맵을 생성한다. 이 과정에서 필터는 입력 채널 수만큼의 깊이를 가진다.
