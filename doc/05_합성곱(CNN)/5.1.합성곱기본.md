# 합성곱 신경망

### 기존 심층 신경망 (DNN)의 이미지 처리 한계점
1. 이미지 데이터는 본질적으로 2차원 또는 3차원 구조를 가지지만, DNN은 이를 1차원 벡터로 변환하여 처리하므로 공간적 정보가 손실된다.
2. 이미지 크기가 커질수록 학습해야 할 파라미터 수가 기하급수적으로 증가해 계산 비용이 커지고 과적합 문제가 발생하기 쉽다.
    - 224 x 224 크기의 RGB 이미지를 처리하는 경우 입력층 뉴런 수만 150,528(224x224x3)개가 필요

### 시각 피질의 국부 수용 영역에서 영감을 받은 합성곱 신경망

### 이미지 처리에서 합성곱 신경망이 가지는 장점
- 합성곱 신경망은 이미지의 2D 구조를 유지해 공간 정보를 잘 보존하며, 파라미터 공유를 통해 학습해야 할 양을 줄여 효율성을 높인다.
- 위치 불변성 덕분에 객체 위치가 달라도 특징을 인식할 수 있어 다양한 변화에 강하며, 컴퓨터 비전 태스크에서 뛰어난 성능을 낸다.

# 합성곱 신경망의 특징

### 지역적 연결성 (Local Connectivity)
- 합성곱 신경망의 각 뉴런은 입력 이미지의 일부 영역에만 연결되어, 지역적 특징을 효과적으로 포착한다.

### 파라미터 공유 (Parameter Sharing)
- 합성곱 신경망의 각 레이어에서는 동일한 필터가 입력 전체에 적용되어, 이미지의 다른 부분에서도 재사용된다. 덕분에 학습해야 할 파라미터 수가 줄고, 특정 패턴을 위치에 상관없이 감지할 수 있다.

### 평행 이동 불변성 (Translation Invariance)
- 합성곱 신경망은 입력 이미지에서 객체 위치가 달라져도 동일한 특징을 감지할 수 있다. 이는 합성곱 연산과 풀링 레이어의 조합 덕분으로, 평행 이동 불변성은 객체가 어디에 있든 인식할 수 있게 해준다.
