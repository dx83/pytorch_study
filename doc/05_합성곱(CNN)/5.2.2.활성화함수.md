# 활성화 함수와 비선형

### ReLU 함수의 특징과 장점
- ReLU (Rectified Linear Unit)는 양수 값은 그대로 유지하고, 음수 값은 0으로 만드는 함수
$$\mathrm{f}(x) = \max(0, x)$$

- 장점
    - 계산이 단순하고 빠름
    - 기울기 소실 문제를 완화
    - 모델에 비선형성을 추가하여 복잡한 패턴을 학습
    - 희소성 (Sparsity)을 촉진하여 모델이 더 효율적으로 학습

<br>

```python
relu = nn.ReLU()
x = torch.linspace(-5, 5, 100)
y = relu(x)

plt.figure(figsize=(8, 4))
plt.plot(x.numpy(), y.numpy())
plt.grid(True)
plt.title('ReLU Activate Function')
plt.xlabel('Input')
plt.ylabel('Output')
plt.axhline(y=0, color='r', linestyle='-', alpha=0.3)
plt.axvline(x=0, color='r', linestyle='-', alpha=0.3)
plt.show()
```
- x가 0보다 작을 때는 출력이 0이고, x가 0보다 클 때는 출력이 x와 같은 직선 형태의 그래프가 그려진다.

<br>

### 기타 활성화 함수들 비교
1. Leaky ReLU: f(x) = max(0.01, x)
    - 음수 입력에 대해 작은 기울기를 갖는다.
    - dying ReLU 문제를 완화한다.

2. ELU(Exponential Linear Unit): f(x) = x if x > 0 else α(e^x - 1)
    - 음수 입력에 대해 부드러운 곡선 형태를 갖는다.
    - 평균 활성화가 0에 가까워 학습 가속화에 도움이 된다.

3. SELU(Scaled Exponential Linear Unit)
    - 자기 정규화 (Self-Normalizing) 특성을 갖는다.
    - 깊은 신경망에서 안정적인 학습을 할 수 있도록 한다.

4. Swish: f(x) = x * sigmoid(x)
    - 구글에서 제안한 활성화 함수로, 일부 태스크에서 ReLU보다 우수한 성능을 보인다.

<br>

```python
#다양한 활성화 함수 비교
x = torch.linspace(-5, 5, 100)
relu = F.relu(x)
leaky_relu = F.leaky_relu(x, 0.01)
elu = F.elu(x)
sigmoid = torch.sigmoid(x)
tanh = torch.tanh(x)
swish = x * torch.sigmoid(x)

plt.figure(figsize=(12, 6))
plt.plot(x.numpy(), relu.numpy(), label='ReLU')
plt.plot(x.numpy(), leaky_relu.numpy(), label='Leaky ReLU')
plt.plot(x.numpy(), elu.numpy(), label='ELU')
plt.plot(x.numpy(), sigmoid.numpy(), label='Sigmoid')
plt.plot(x.numpy(), tanh.numpy(), label='Tanh')
plt.plot(x.numpy(), swish.numpy(), label='Swish')
plt.grid(True)
plt.legend()
plt.title('Comparison of different activation functions')
plt.xlabel('Input')
plt.ylabel('Output')
plt.axhline(y=0, color='r', linestyle='-', alpha=0.3)
plt.axvline(x=0, color='r', linestyle='-', alpha=0.3)
plt.show()
```

<br>

### 합성곱 신경망에서의 활성화 함수 선택 기준
1. 계산 효율성: 모델의 학습 및 추론 속도에 영향을 미친다.
2. 기울기 소실 문제: 깊은 네트워크에서 기울기가 사라지지 않는 함수가 유리하다.
3. 비선형성: 복잡한 패턴을 학습하기 위해 충분한 비선형성을 제공해야 한다.
4. 태스크 특성: 분류, 회귀 등 태스크에 따라 적합한 함수가 다를 수 있다.
- 일반적으로 은닉층에서는 ReLU나 그 변형을 사용하고, 출력층에서는 분류에는 Softmax, 회귀에는 선형 함수 등을 사용한다.
