# 풀링 레이어 (Pooling Layer)

### 최대 풀링과 평균 풀링
- 풀링 레이어는 특성 맵의 공간적 크기를 줄이는 다운 샘플링 연산을 수행한다.
    - 최대 풀링 (Max Pooling): 윈도우 내에서 가장 큰 값을 선택
    - 평균 풀링 (Average Pooling): 윈도우 내 값들의 평균을 계산

```python
입력 특성 맵:
   3       -1     12     -1
  ━━━━━  ━━━━━  ━━━━━  ━━━━━
   -3       1      0      1
  ─────  ─────  ─────  ─────
   2       -3      0      1
  ─────  ─────  ─────  ─────
   3       -2      4     -1
```

최대 풀링 출력:
$$
\begin{bmatrix}
3 & 12\\
3 & 4
\end{bmatrix}
$$

평균 풀링 출력:
$$
\begin{bmatrix}
0 & 3\\
0 & 1
\end{bmatrix}
$$

<br>

### 풀링의 목적과 효과
- 파라미터 수 감수: 특성 맵의 크기를 줄여 계산 효율성을 높인다.
- 과적합 방지: 모델의 복잡도를 줄여 과적합 위험을 감소시킨다.
- 위치 불변성 향상: 특성의 정학한 위치보다 존재 여부가 중요한 경우에 유용하다.
- 수용 영역 (Receptive Field) 확대: 상위 레이어의 뉴런이 더 넓은 입력 영역을 볼 수 있게 한다.

<br>

### 풀링 레이어 설게 시 고려 사항
- 풀링 크기: 보통 2x2가 많이 사용, 더 큰 크기는 정보 손실이 커질 수 있다.
- 스트라이드: 보통 풀링 크기와 동일하게 설정
- 풀링 유형 선택: 최대 풀링은 텍스처나 패턴 감지에 좋고, 평균 풀링은 배경 정보 보존에 유리
- 풀링 레이어 수: 너무 많은 풀링 레이어는 중요한 공간 정보를 과도하게 손실할 수 있다.

<br>

`전역 평균 풀링 (Global Average Pooling)`
- 각 특성 맵을 하나의 값으로 축소하여 완전 연결 레이어의 필요성을 줄이고 과적합을 방지하는데 도움

```python
import torch
import torch.nn as nn

# 샘플 특성 맵 (배치 크기 1, 채널 3, 높이 4, 너비 4)
feature_maps = torch.randn(1, 3, 4, 4)

# 전역 평균 풀링 적용
global_avg_pool = nn.AdaptiveAvgPool2d((1, 1))
output = global_avg_pool(feature_maps)
print(f"원본 특성 맵 크기: {feature_maps.shape}")       # [1, 3, 4, 4]
print(f"전역 평균 풀링 후 크기: {output.shape}")         # [1, 3, 1, 1]
print(f"평탄화 후 크기: {output.view(1, -1).shape}")    # [1, 3]
```
