### Tensor 와 NumPy 배열 비교
- 유사점
    - 다차원 배열 구조
    - 일관된 데이터 타입 유지
    - 다양한 수학 연산 지원

- 차이점
    - GPU 활용
        - 파이토치 텐서 : `.to('장치명')` 메서드를 사용하여 텐서를 GPU로 쉽게 이동
        - NumPy 배열 : 기본으로 CPU 연산, GPU 가속 기능 없음
    - 자동 미분
        - 파이토치 텐서 : `requires_grad=True` 사용시 연산 과정을 추적하고, 역전파할 때 기울기(gradient)를 계산
        - NumPy 배열 : 자동 미분 기능 없음
    - 딥러닝 특화
        - 파이토치 텐서 : 손실 함수, 옵티마이저, 네트워크 레이어 등 고급 기능 제공
        - NumPy 배열 : 일반적인 과학 계산과 데이터 처리를 위한 라이브러리, 딥러닝에 최적화된 기능 없음

<br>

### 텐서와 넘파이 간 상호 변환
```pyton
import numpy as np
import torch
```

1. NumPy 배열 => 파이토치 텐서
```pyton
np_arr = np.array([1,2,3])
tensor_from_np = torch.from_numpy(np_arr)
```

2. 파이토치 텐서 => NumPy 배열
```python
np_from_tensor = tensor_from_np.numpy()
```

3. 주의 사항
- CPU 텐서와 NumPy 배열의 메모리 공유
    - 파이토치의 텐서를 NumPy 배열로 변환하면 두 객체는 동일한 메모리 블록을 참조
- GPU 텐서와 NumPy 배열
    - GPU에 위치한 텐서는 직접 NumPy 배열로 변환 불가
    - .cpu() 메서드로 텐서를 CPU 메모리로 이동해야 함
