# PyTorch 텐서 차원 조작 정리

## 1. transposed / transpose

- 2차원 텐서에서 행과 열을 서로 바꾸거나, 지정한 두 차원의 위치를 교환한다.

```python
import torch

x = torch.tensor([
    [1, 2, 3],
    [4, 5, 6]
])

print(x.shape)      # torch.Size([2, 3]) : 2행 3열
print(x.t())        # 2차원 텐서 전용 전치, 행과 열을 바꿈
print(x.t().shape)  # torch.Size([3, 2]) : 3행 2열

print(x.transpose(0, 1))  # 0번 차원(행)과 1번 차원(열)을 교환
```

`t()`는 2차원 텐서에서만 사용하는 간단한 전치 함수이고, `transpose(dim0, dim1)`는 두 차원을 지정해서 서로 바꿀 수 있다.

<br>

## 2. permute

- 3차원 이상 텐서에서 전체 차원의 순서를 원하는 순서로 재배열한다.

```python
import torch

x = torch.randn(2, 3, 4)  # 모양: [배치, 높이, 너비]라고 가정

print(x.shape)  # torch.Size([2, 3, 4])

y = x.permute(0, 2, 1)  # 0번 차원은 그대로, 2번 차원과 1번 차원의 순서를 바꿈

print(y.shape)  # torch.Size([2, 4, 3])
```

`transpose()`는 두 차원만 교환하지만, `permute()`는 모든 차원의 순서를 한 번에 새로 지정한다.

예를 들어 이미지 텐서에서 자주 쓰는 변환은 다음과 같다.

```python
import torch

image = torch.randn(224, 224, 3)  # [높이, 너비, 채널], 일반 이미지 형태

image_for_pytorch = image.permute(2, 0, 1)  # [채널, 높이, 너비]로 변경

print(image_for_pytorch.shape)  # torch.Size([3, 224, 224])
```

## 3. squeeze

- 크기가 1인 차원을 제거해서 텐서의 차원 수를 줄인다.

```python
import torch

x = torch.randn(1, 3, 1, 5)

print(x.shape)  # torch.Size([1, 3, 1, 5])

y = x.squeeze()  # 크기가 1인 모든 차원을 제거

print(y.shape)  # torch.Size([3, 5])
```

특정 위치의 차원만 제거할 수도 있다.

```python
import torch

x = torch.randn(1, 3, 1, 5)

y = x.squeeze(0)  # 0번 차원의 크기가 1이므로 제거됨

print(y.shape)  # torch.Size([3, 1, 5])

z = x.squeeze(1)  # 1번 차원의 크기는 3이므로 제거되지 않음

print(z.shape)  # torch.Size([1, 3, 1, 5])
```

`squeeze(dim)`은 해당 차원의 크기가 1일 때만 제거하고, 크기가 1이 아니면 모양이 그대로 유지된다.

<br>

## 4. unsqueeze

- 지정한 위치에 크기가 1인 새 차원을 추가해서 텐서의 차원 수를 늘린다.

```python
import torch

x = torch.tensor([1, 2, 3])

print(x.shape)  # torch.Size([3]) : 1차원 텐서

y = x.unsqueeze(0)  # 0번 위치에 새 차원 추가

print(y.shape)  # torch.Size([1, 3]) : 행이 1개인 2차원 텐서
print(y)        # tensor([[1, 2, 3]])

z = x.unsqueeze(1)  # 1번 위치에 새 차원 추가

print(z.shape)  # torch.Size([3, 1]) : 열이 1개인 2차원 텐서
print(z)
```

모델 입력에 배치 차원을 추가할 때 자주 사용한다.

```python
import torch

image = torch.randn(3, 224, 224)  # [채널, 높이, 너비] 형태의 이미지 1장

batch = image.unsqueeze(0)  # 맨 앞에 배치 차원 추가

print(batch.shape)  # torch.Size([1, 3, 224, 224])
```

## 핵심 비교

| 함수 | 역할 | 주 사용 상황 |
|---|---|---|
| `t()` | 2차원 텐서의 행과 열을 바꿈 | 행렬 전치 |
| `transpose(dim0, dim1)` | 두 차원을 서로 교환 | 2차원 또는 고차원 텐서에서 두 축만 바꿀 때 |
| `permute(...)` | 전체 차원 순서를 재배열 | 이미지 텐서의 `[H, W, C]`를 `[C, H, W]`로 바꿀 때 |
| `squeeze()` | 크기가 1인 차원 제거 | 불필요한 배치/채널 차원 제거 |
| `unsqueeze(dim)` | 크기가 1인 차원 추가 | 배치 차원 추가 |
