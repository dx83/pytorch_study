# 순전파 (Forward Propagation)
- 입력층에서 시작된 데이터는 가중치와 편향을 적용한 후 활성화 함수를 거쳐 출력층에 도달하는 과정
- 예시) 입력 이미지 -> 은닉충 -> 출력층 -> 예측 결과
- `입력을 넣어서 예측하는 과정`

<br>

- 가중치 행렬 W, 입력 벡터 x, 편향 벡터 b
```
z = Wx + b
```

<br>

# 역전파 (Backpropagation)
- 예측값이 정답과 얼마나 다른지 계산한 뒤, 그 오차를 뒤에서 앞으로 보내면서 가중치를 수정하는 과정
- `틀린 정도를 보고 가중치를 고치는 과정`

<br>

# 손실 함수와 옵티마이저

### 평균 제곱 오차(Mean Squared Error, MSE)
- 회귀 문제에서 사용, 모델의 예측값과 실제 정답 간 차이를 제곱하여 평균을 구하는 방식으로 오차를 측정
- 심층 신경망에서는 기울기가 작은 경우 학습 속도가 느려질 수 있으며, 이를 해결하기 위해 MAE(Mean Absolute) 또는 Huber 손실을 사용하기도 함

### 교차 엔트로피 (Cross-Entropy)
- 분류 문제에서 사용, 모델의 예측 확률과 실제 정답 간의 차이를 측정
- 분류 문제의 유혀에 따라 다른 형태를 사용
    - 이진 분류 (Binary Classification) : 2개의 클래스를 구분하는 문제
        - 스팸/정상 메일, 고양이/개
        - 출력층에 Sigmoid 함수 사용
    - 다중 분류 (Multi-Classification) : 3개 이상의 클래스를 구분하는 문제
        - MNIST 숫자 분류, 이미지 분류
        - 원-핫 인코딩된 정답 벡터
        - 출력층에 Softmax 함수 사용
        - 각 샘플은 하나의 클래스에만 속함

### 확률적 경사 하강법 (Stochastic Gradient Descent, SGD)
- 반복마다 무작위 샘플을 선택하여 가중치를 업데이트하는 방식, 전체 데이터셋을 사용하지 않고 일부 샘플만을 이용해 학습을 진행
- 확률적 방식으로 학습이 이루어지므로 지역 최솟값(Local Minimum)을 벗어나 전역 최솟값(Global Minimum)에 도달할 가능성이 높음
- 그러나 노이즈가 많아 학습 속도가 불안정할 수 있으며, 이를 개선하기 위해 미니 배치 SGD(Mini-Batch SGD)가 자주 사용

### Adam 옵티마이저 (Adaptive Moment Estimation)
- SGD의 단점을 보완하기 위하여 개발된 옵티마이저로, 학습률을 동적으로 조정하는 방식으로 빠르고 안정적인 수렴을 제공
- Gradient 평균과 Gradient 제곱 평균을 동시에 추정해 가중치를 업데이트하는 방식
- Adam은 학습 속도가 빠르고, 학습률을 개별 가중치에 맞게 조정해 최적화가 원활히 진행되도록 도와준다.

### 학습률 (Learning Rate)
- 신경망의 학습 속도를 조절하는 중요한 하이퍼파라미터
- 학습률 스케줄링(Learning Rate Scheduling) 기법을 적용하면 학습이 진행됨에 따라 학습률을 동적으로 변경해 학습 성능을 개선할 수 있다.

### 모멘텀 (Momentum)
- 이전 기울기 값을 고려하여 학습 방향을 조정해 수렴 속도를 개선하는 기법
- 경사 하강법에서 발생할 수 있는 진동(Oscillation) 문제를 완화하고, 더 빠르게 최적해에 도달할 수 있도록 돕는다.
- Nesterov Momentum은 기존 모멘텀보다 더 빠르고 효율적인 학습을 유도하며, 이는 미래의 기울기를 미리 예측하여 업데이트를 수행하는 방식이다.
