# 신경망의 기본 구성 요소

### 입력 (Input)
- 외부 데이터 또는 이전 층의 뉴런에서 출련된 값을 받는 역할

### 가중치 (Weight)
- 입력값의 중요도를 결정하며, 학습 과정에서 최적의 값을 찾는다.

### 편향 (Bias)
- 모델의 유연성을 높이고 더 정확한 학습을 할 수 있도록 하는 상수값

### 활성화 함수 (Activation Function)
- 선형 연산을 비선형 변환으로 변형해 복잡한 패턴을 학습할 수 있도록 한다.
- ReLU, Sigmoid, Tanh

### 출력 (Output)
- 연산 괄과를 다음 층으로 전달하거나 최종 결과로 출력

<br>

# 신경망의 종류

### 다층 퍼셉트론 (MLP, Multi-Layer Perceptron)
- 인공 신경망의 기본적인 형태, 구조화된 정형 데이터에 적합
- 여러 개의 은닉층(Hidden Layer)으로 구성된 완전 연결 신경망(Fully Connected Neural Network)
- 이미지나 시계열 데이터 처리에 적합하지 않다.

### 합성곱 신경망 (CNN, Convolutional Neural Network)
- 이미지 데이터와 같은 2차원의 구조화된 데이터에 적합하도록 설계된 신경망
- 합성곱 연산 (Convolution)과 풀링 (Pooling) 연산을 활용하여 특징을 추출
- 컴퓨터 비전 (이미지 분류, 객체 탐지, 영상 분할 등)에 널리 활용

### 순환 신경만 (RNN, Recurrent Neural Network)
- 시계열 데이터 또는 순차적 데이터를 다루는 데 특화된 신경망 구조
- 이전 상태의 정보를 저장해 시간에 따라 변하는 데이터를 효과적으로 처리
- 장기 의존성 문제 - 긴 문맥을 기억하는 데 약함
- 자연어 처리 (기계 번여, 감성 분석, 음성 인식등)에 적용

# 신경망 선택 가이드 라인

### 입력 데이터의 형태 분석
- 정형 데이터(Structured Data) : MLP
- 이미지 데이터 : CNN
- 시계열 데이터 또는 순차 데이터 (Sequential Data) : RNN, Transformer

### 데이터의 크기와 연산량 고려
- 소규모 데이터셋: MLP, CNN
- 대규모 데이터셋: 심층 신경망 (Deep Neural Network), 사전 학습된 모델 (Pretrained Model)

### 실시간 응용 여부
- 가벼운 모델 구조를 채택하고 최적화 기법 (양자화 Quantization, 가지치기 Pruning) 적용
- 정확성이 중요한 경우 대규모 모델을 활용하되 계산 성능을 고려하여 배포 전략 수립

### 과적합 방지 전략
- 정규화(Normalization), 드롭아웃 (Dropout)을 적용
- 적절한 데이터 증강 (Data Augmentation) 기법을 사용해 학습 데이터의 다양성 확보

### 최신 연구 동향 반영
- CNN => ViT (Vision Transformer) 로 전환
- RNN 기반 NLP => 트랜스포머 기반 BERT, GPT 등으로 변화
- 온디바이스 AI
