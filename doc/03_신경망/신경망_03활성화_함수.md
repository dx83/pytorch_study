# 활성화 함수 (Activation Function)

### Sigmoid 함수
- 출력 범위를 (0, 1)로 제한하여 확률적 해석을 할 수 있다.
- 입력값이 매우 크거나 작을 경우 기울기가 0에 가까워지는 기울기 소실 (Vanishing Gradient) 문제 
- 이진 분류 문제에 널리 사용, 깊은 신경망에서는 잘 사용되지 않음

### ReLU (Rectified Linear Unit) 함수
- 계산량이 적고 학습 속도가 빠름, 가장 널리 사용되는 활성화 함수
- 입력이 0 이하일 때는 0을 출력하고 양수일 때는 그대로 출력
- Sigmoid 함수보다 학습이 빠르고, 깊은 신경망에서 기울기 소실 문제 완화
- 입력값이 0 이하일 때 뉴런이 완전히 죽어 버리는 문제 (Dying ReLU)가 발생

### Tanh 함수
- Sigmoid와 유사하지만, 출력 범위가 (-1, 1)로 설정되어 있어 중심이 0인 데이터를 학습하는 데 유리
- 기울기 소실 문제, 깊은 신경망에서 정보가 손실될 가능성
- 은닉층에서 데이터 분포를 조절하는 역할을 할 수 있어 일부 모델에서 유용
- 일부 비선형 패턴을 학습하는 데 유리

# 활성화 함수 선택 기준
- 일반적으로 은닉층에서는 ReLU 사용
- 분류 문제 Softmax 함수, 회귀 문제 Linear 함수 사용
