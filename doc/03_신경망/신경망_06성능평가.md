# 성능 평가 지표

### 정확도 (Accuracy)
- 전체 샘플 중에서 올바르게 분류된 샘플의 비율
$$
\text{Accuracy} = \frac{TP + TN}{TP + TN + FP + FN}
$$
> `TP`: 실제 Positive를 Positive로 예측  
> `TN`: 실제 Negative를 Negative로 예측  
> `FP`: 실제 Negative를 Positive로 잘못 예측  
> `FN`: 실제 Positive를 Negative로 잘못 예측  

> `TP + TN`은 올바르게 예측한 샘플 수이고,  
> `TP + TN + FP + FN`은 전체 샘플 수이다.

<br>

### 정밀도 (Precision)와 재현율 (Recall)
- 정밀도 : 모델이 긍정 클래스로 예측한 샘플 중 실제로 긍정 클래스인 비율
$$
\text{Precision} = \frac{TP}{TP + FP}
$$

- 재현율 : 실제 긍정 클래스 샘플 중에서 모델이 올바르게 예측한 비율
$$
\text{Recall} = \frac{TP}{TP + FN}
$$

- 정밀도와 재현율은 서로 상충 관계, 특정 문제에서는 한 쪽을 더욱 중요하게 고려해야 한다.

<br>

### 혼동 행렬 (Confusion Matrix) 분석
- 분류 모델의 성능을 시각적으로 분석하기 위해 사용
- True Positive (TP), True Negative (TN), False Positive (FP), False Negative (FN)

```python
cm_mnist = confusion_matrix(y_true, y_pred)
plt.figure(figsize=(10, 8))
sns.heatmap(cm_mnist, annot=True, fmt='d', cmap='Blues',
            xticklabels=range(10), yticklabels=range(10))
plt.xlabel('Predicted')
plt.ylabel('Actual')
plt.title('MNIST Confusion Matrix')
plt.show()
```

<br>

### ROC 커브 (Receiver Operating Characteristic Curve) 해석
- 모델의 민감도 (Recall)와 특이도(Specificity) 간 관계를 나타냄 
- AUC-ROC 값이 1에 가까울수록 성능이 우수
- AUC (Area Under the Curve) : ROC 커브 아래 면적

```python
y_scores = np.array([0.1, 0.4, 0.35, 0.8])
y_test = np.array([0, 0, 1, 1])

fpr, tpr, _ = roc_curve(y_test, y_scores)
roc_auc = auc(fpr, tpr)

plt.figure(figsize=(8,6))
plt.plot(fpr, tpr, label=f'ROC curve (area = {roc_auc:.2f})')
plt.xlabel('False Positive Rate')               # FPR, 거짓 양성 비율
plt.ylabel('True Positive rate')                # TPR, 재현율
plt.title('Receiver Operating Characteristic')
plt.legend()
plt.grid(True)
plt.show()
```
