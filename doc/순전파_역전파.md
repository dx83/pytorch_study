# 순전파 (Forward Propagation)
- 입력층에서 시작된 데이터는 가중치와 편향을 적용한 후 활성화 함수를 거쳐 출력층에 도달하는 과정
- 예시) 입력 이미지 -> 은닉충 -> 출력층 -> 예측 결과
- `입력을 넣어서 예측하는 과정`

<br>

- 가중치 행렬 W, 입력 벡터 x, 편향 벡터 b
```
z = Wx + b
```

<br>

# 역전파 (Backpropagation)
- 예측값이 정답과 얼마나 다른지 계산한 뒤, 그 오차를 뒤에서 앞으로 보내면서 가중치를 수정하는 과정
- `틀린 정도를 보고 가중치를 고치는 과정`

<br>
